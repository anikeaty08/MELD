# MELD Docs App

This folder contains the standalone React documentation site for MELD.

## Run locally

```bash
cd docs-site
npm install
npm run dev
```

The Vite dev server runs on `http://localhost:4173` by default.

## Build

```bash
cd docs-site
npm run build
```

## What this docs app covers

- installation with full `pip` commands
- Windows, macOS, and Linux setup
- benchmark quickstart commands
- custom dataset provider training
- direct `DeltaModel` training with manual `DataLoader`s
- architecture, metrics, and research chapters
